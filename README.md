# 9 Personalities of Yunnan Ethnics (9PYE)

## 项目简介

9 Personalities of Yunnan Ethnics (9PYE) 是一个创新的在线心理测试平台，巧妙地将现代人格心理学测试与云南丰富多彩的少数民族文化相结合。用户只需10分钟即可获得一份详细而深刻的个性化报告，不仅能准确分析用户的人格特质，还能精准匹配相应的云南少数民族类型。

## 🌟 特色功能

### **人格测试系统**
- 基于心理学理论设计的综合性测试问卷
- 10分钟快速测试，即时生成个性化报告
- 精准分析用户的人格特质和行为模式

### **民族文化匹配**
- 覆盖云南9个主要少数民族：白族、哈尼族、傣族、傈僳族、拉祜族、佤族、纳西族、布朗族、独龙族
- 每个民族都有独特的人格画像和文化内涵
- 深度解读用户与对应民族的文化共鸣

### **文化内容展示**
- **服饰文化**：展示各民族传统服饰的特色与寓意
- **语言文化**：介绍各民族的语言文字和表达方式
- **小剧场**：通过情景剧形式展现民族文化特色
- **民族百科**：详细的历史背景和文化传承
